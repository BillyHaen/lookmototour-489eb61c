# Project Memory

## Core
React + Vite ONLY. No Next.js or other frameworks.
Indonesian UI, mobile-first layout. Moto-touring aesthetic (Navy Blue & Sky Blue).
Supabase backend: Use strict RLS, soft-deletes, and RPCs for public profile data.
WhatsApp is the primary channel for direct communication and shop checkout.
Use the custom LookMotoTour logo for branding, no generic icons.
